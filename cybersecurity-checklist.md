# 개인 사이버보안 체크리스트

## 일상 보안 수칙

### 패스워드 관리
- [ ] 복잡한 패스워드 사용 (8자 이상, 영문/숫자/특수문자 조합)
- [ ] 각 사이트마다 다른 패스워드 사용
- [ ] 패스워드 매니저 사용

### 소프트웨어 보안
- [ ] 운영체제 자동 업데이트 활성화
- [ ] 안티바이러스 프로그램 설치 및 실시간 스캔 활성화
- [ ] 신뢰할 수 있는 소스에서만 소프트웨어 다운로드

### 네트워크 보안
- [ ] 공공 Wi-Fi 사용 시 주의
- [ ] VPN 사용 (필요시)
- [ ] 홈 네트워크 라우터 기본 패스워드 변경
